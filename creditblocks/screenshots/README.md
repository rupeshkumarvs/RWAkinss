# Screenshots

This folder contains screenshots for the CreditBlocks project.

## Required Screenshots

1. **landing.png** - Landing page with wallet connection
2. **generate_passport.png** - Score generation interface showing score, risk band, and transaction hash
3. **explorer_tx.png** - QIE Explorer showing the on-chain transaction

## How to Add Screenshots

1. Take screenshots of your application
2. Save them with the names above
3. Add them to this folder
4. Update README.md to reference these images

Example markdown:
```markdown
![Landing Page](./screenshots/landing.png)
![Generate Passport](./screenshots/generate_passport.png)
![Explorer Transaction](./screenshots/explorer_tx.png)
```
