# SyncSplit — User Feedback Form (Google Forms Template)

Use this structure to create a Google Form for collecting user feedback.

---

## Form Title
**SyncSplit User Feedback — Stellar Testnet**

## Form Description
Thank you for testing SyncSplit! This feedback helps us improve the protocol.
All responses are anonymous unless you choose to share your wallet address.

---

## Fields

### 1. Name
- **Type**: Short answer
- **Required**: No
- **Placeholder**: "Your name or alias"

### 2. Wallet Address
- **Type**: Short answer
- **Required**: Yes
- **Validation**: Starts with `G` and is 56 characters
- **Placeholder**: "GABC...XYZ (your Stellar public key)"
- **Helper text**: "Your Stellar testnet public key (starts with G)"

### 3. Overall Rating
- **Type**: Linear scale
- **Required**: Yes
- **Scale**: 1 to 5
- **Label (Low)**: "Poor"
- **Label (High)**: "Excellent"

### 4. What did you like most?
- **Type**: Paragraph
- **Required**: No
- **Placeholder**: "e.g., The UI design, ease of splitting, transaction speed..."

### 5. Issues or bugs encountered
- **Type**: Paragraph
- **Required**: No
- **Placeholder**: "e.g., Wallet didn't connect, transaction failed, confusing UI..."

### 6. Feature requests
- **Type**: Paragraph
- **Required**: No
- **Placeholder**: "e.g., Support for custom tokens, group chat, payment reminders..."

### 7. Would you use SyncSplit again?
- **Type**: Multiple choice
- **Required**: Yes
- **Options**:
  - Yes, definitely
  - Maybe
  - No

### 8. How would you rate the onboarding experience?
- **Type**: Linear scale
- **Required**: No
- **Scale**: 1 to 5
- **Label (Low)**: "Very confusing"
- **Label (High)**: "Very smooth"

---

## After Creating the Form

1. Copy the form URL
2. Add it to `.env` as `VITE_FEEDBACK_FORM_URL=https://forms.google.com/...`
3. The onboarding stepper's final step will link to it
4. To parse responses, export as CSV from Google Sheets and run:

```bash
node scripts/parse_feedback.mjs path/to/responses.csv
```

## CSV Column Headers Expected

The CSV parser expects these exact column headers (auto-generated by Google Forms):

```
Timestamp, Name, Wallet Address, Overall Rating, What did you like most?, Issues or bugs encountered, Feature requests, Would you use SyncSplit again?, How would you rate the onboarding experience?
```
